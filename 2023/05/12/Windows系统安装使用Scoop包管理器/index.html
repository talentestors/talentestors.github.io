Windowsзі»з»ҹе®үиЈ…дҪҝз”ЁScoopеҢ…з®ЎзҗҶеҷЁ

stazxr В· 2023-5-12 В· ж¬Ўйҳ…иҜ»

еүҚиЁҖ

ScoopжҳҜWindowsзҡ„е‘Ҫд»ӨиЎҢе®үиЈ…зЁӢеәҸгҖӮ

еҰӮжһңз”ЁиҝҮLinuxзі»з»ҹпјҢдҪҝз”Ёapt-getе·Ҙе…·е®үиЈ…иҝҮиҪҜд»¶пјҢжҲ–иҖ…з”ЁиҝҮPythonпјҢзҹҘйҒ“pipе·Ҙе…·з”ЁдәҺз®ЎзҗҶPythonеҗ„з§Қдҫқиө–еҢ…пјҢйӮЈд№ҲзҗҶи§ЈScoopе°ұжҜ”иҫғе®№жҳ“пјҢиҝҷдәӣе·Ҙе…·зҡ„и®ҫи®ЎзҗҶеҝөдёҺдҪҝз”Ёж–№жі•йғҪйқһеёёзұ»дјјгҖӮ

ScoopдёҖиҲ¬з”ЁжқҘз®ЎзҗҶз»ҝиүІиҪҜд»¶пјҢеҚіжҳҜдёҖз§ҚйҖҡиҝҮи§ЈеҺӢеҺӢзј©еҢ…еҚіеҸҜе°ұең°дҪҝз”Ёзҡ„иҪҜд»¶пјҢеҜ№дәҺжҜ”иҫғдё“дёҡиҪҜд»¶пјҲжҜ”еҰӮOfficeгҖҒAdobeзӯүпјүж”ҜжҢҒдёҚеҘҪ

еҹәзЎҖзҺҜеўғй…ҚзҪ®

еңЁеҮҶеӨҮе®үиЈ…д№ӢеүҚпјҢе»әи®®иҝӣиЎҢй…ҚзҪ®пјҲеҰӮжһңдҪ иғҪи§ЈеҶіGitHubи®ҝй—®й—®йўҳпјҢиҜ·и·іиҝҮрҹҳ„пјү

еҰӮжһңдҪ жҳҜwin10зі»з»ҹпјҢе»әи®®е®үиЈ…TerminalпјҲwin11е·Із»ҸиҮӘеёҰпјҢиҜ·и·іиҪ¬иҮіжӯЈејҸејҖе§ӢScoopзҡ„е®үиЈ…пјү

ж–°дёҖд»Јз»Ҳз«ҜWindows Terminal
ж”ҜжҢҒж ҮзӯҫйЎөгҖҒеӨҡз§Қе‘Ҫд»ӨиЎҢе·Ҙе…·еӨҡејҖвҖҰ
еҸҜе®ҡеҲ¶зҫҺеҢ–

Terminal з»Ҳз«Ҝ

е®үиЈ… Terminal

  1. жү“ејҖWindowsиҮӘеёҰзҡ„еә”з”Ёе•Ҷеә—пјҲMicroSoft Storeпјү
  2. жҗңзҙўTerminal
  3. е®үиЈ…

е»әи®®ж·»еҠ д»»еҠЎж Ҹж–№дҫҝдҪҝз”Ё

Terminal дҪҝз”Ё

йҰ–ж¬ЎиҝҗиЎҢж—¶пјҢWindows Terminalд»ҘPowershellдҪңдёәй»ҳи®Өй…ҚзҪ®ж–Үд»¶еҗҜеҠЁгҖӮдёӢжӢүиҸңеҚ•еҸҜз”ЁдәҺеҗҜеҠЁе…¶д»–йҖүйЎ№еҚЎе№¶и®ҝй—®и®ҫзҪ®пјҡ

еҸҜд»ҘеңЁи®ҫзҪ®йҮҢйқўиҝӣиЎҢдҝ®ж”№йҖҸжҳҺеәҰпјҢжӣҙж”№иғҢжҷҜзӯүи®ҫзҪ®

жҲ‘зҡ„дёӘдәәж•ҲжһңеҰӮдёӢпјҡ

жӣҙеӨҡдҪҝз”ЁжҠҖе·§иҜ·зӮ№еҮ»и®ҝй—®е®ҳж–№ж–ҮжЎЈ

жӯЈеёёи®ҝй—® GitHub

з”ұдәҺGitHubеңЁеӣҪеҶ…ж—¶еёёдјҡиў«DNSеҠ«жҢҒпјҢжүҖд»ҘйңҖиҰҒиҝӣиЎҢжң¬ең°еҸҚд»ЈзҗҶпјҢдҝқиҜҒи®ҝй—®GitHubзЁіе®ҡгҖӮ

жң¬ж–ҮжҺЁиҚҗдҪҝз”ЁSteam++жқҘе®ҢжҲҗжң¬ең°еҸҚд»ЈгҖӮ

е®үиЈ…пјҡ

  1. жү“ејҖWindowsиҮӘеёҰзҡ„Microsoft Store.
  2. жҗңзҙўWatt Toolkitе№¶е®үиЈ….

з„¶еҗҺпјҢз»ҷGitHubжү“дёҠеӢҫзӮ№дёҖй”®еҠ йҖҹ.

жү“ејҖGitHubй“ҫжҺҘзңӢзңӢжҳҜеҗҰиғҪи®ҝй—®пјҡhttps://github.com

жӯЈејҸејҖе§ӢScoopзҡ„е®үиЈ…

githubең°еқҖпјҡhttps://github.com/ScoopInstaller/Scoop

ж–ҮжЎЈең°еқҖпјҡhttps://scoop.sh/

е‘Ҫд»Өе®үиЈ…

дёәдәҶи®©PowerShellеҸҜд»Ҙжү§иЎҢи„ҡжң¬пјҢйҰ–е…ҲйңҖиҰҒи®ҫзҪ®PowerShellжү§иЎҢзӯ–з•ҘпјҢйҖҡиҝҮиҫ“е…Ҙд»ҘдёӢе‘Ҫд»ӨSet-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUserпјҲеңЁд№ӢеүҚе®үиЈ…еҘҪзҡ„Terminalжү§иЎҢпјү

д»Һйқһз®ЎзҗҶе‘ҳ PowerShell иҝҗиЎҢжӯӨе‘Ҫд»Өд»ҘдҪҝз”Ёй»ҳи®Өй…ҚзҪ®е®үиЈ… scoopпјҢ Scoopе°Ҷе®үиЈ…еҲ° C:\Users<YOUR USERNAME>\scoop

irm get.scoop.sh | iex
# дҪҝз”Ёй•ңеғҸе®үиЈ…
irm get.scoop.sh -Proxy 'http://<ip:port>' | iex

жҲ–иҖ…д»ҘдёӢе‘Ҫд»Өпјҡ

Invoke-Expression (New-Object System.Net.WebClient).DownloadString('https://get.scoop.sh')

й«ҳзә§е®үиЈ…

еҰӮжһңдҪ дёҚжғіе®үиЈ…еҲ°CзӣҳпјҢеҸҜйҮҮз”ЁжӯӨж–№жі•пјҲе°ҸзҷҪдёҚжҺЁиҚҗпјү

е…Ҳжү§иЎҢ

irm get.scoop.sh -outfile 'install.ps1'

дёӢиҪҪе®үиЈ…зЁӢеәҸ

еҸҜд»Ҙexplorer .еңЁеҪ“еүҚжү“ејҖиө„жәҗз®ЎзҗҶеҷЁпјҢжҹҘзңӢдёӢиҪҪзҡ„ж–Үд»¶

жҹҘзңӢе®үиЈ…зЁӢеәҸзҡ„жүҖжңүеҸҜй…ҚзҪ®еҸӮж•°гҖӮ

.\install.ps1 -?

дҫӢеҰӮпјҢеҸҜд»Ҙе°Ҷ scoop е®үиЈ…еҲ°иҮӘе®ҡд№үзӣ®еҪ•пјҢй…ҚзҪ® scoop иҝӣиЎҢе®үиЈ…е…ЁеұҖзЁӢеәҸеҲ°иҮӘе®ҡд№үзӣ®еҪ•пјҢе№¶еңЁе®үиЈ…иҝҮзЁӢдёӯз»•иҝҮзі»з»ҹд»ЈзҗҶгҖӮ

.\install.ps1 -ScoopDir 'D:\Applications\Scoop' -ScoopGlobalDir 'F:\GlobalScoopApps' -NoProxy

жҲ–иҖ…пјҢеҸҜд»ҘдҪҝз”Ёж—§ж–№жі•йҖҡиҝҮи®ҫзҪ®зҺҜеўғеҸҳйҮҸжқҘй…ҚзҪ®иҮӘе®ҡд№үзӣ®еҪ•гҖӮпјҲдёҚжҺЁиҚҗпјү

$env:SCOOP='D:\Applications\Scoop'
$env:SCOOP_GLOBAL='F:\GlobalScoopApps'
[Environment]::SetEnvironmentVariable('SCOOP_GLOBAL', $env:SCOOP_GLOBAL, 'Machine')
irm get.scoop.sh | iex

й»ҳи®Өжғ…еҶөдёӢе·ІзҰҒз”Ёз®ЎзҗҶе‘ҳжҺ§еҲ¶еҸ°дёӢScoopзҡ„е®үиЈ…пјҢеҰӮжһңйңҖиҰҒдҪҝз”Ё -RunAsAdmin

irm get.scoop.sh -outfile 'install.ps1'
.\install.ps1 -RunAsAdmin [-OtherParameters ...]
# I don't care about other parameters and want a one-line command
iex "& {$(irm get.scoop.sh)} -RunAsAdmin"

Scoopй…ҚзҪ®

еҰӮжһңиҫ“еҮәдәҶsuccessfullyпјҢе°ұд»ЈиЎЁе·ІжҲҗеҠҹе®үиЈ…ScoopгҖӮ

жү“ејҖC:\Users\<user name>\scoopи·Ҝеҫ„е°ұеҸҜд»ҘзңӢеҲ°дёӢйқўзҡ„ж–Үд»¶еӨ№

  • app вҖ”вҖ” е®үиЈ…зҡ„иҪҜд»¶ж”ҫеңЁиҝҷйҮҢ
  • bucket вҖ”вҖ” е·Із»Ҹж·»еҠ зҡ„иҪҜд»¶д»“еә“
  • shims вҖ”вҖ” ScoopеҸҜд»ҘиҮӘеҠЁз”ҹжҲҗshellи„ҡжң¬пјҢзңҒеҺ»дәҶWindowsй…ҚзҺҜеўғеҸҳйҮҸзҡ„зғҰжҒјпјҲжҲ‘зҡ„jdkпјҢMG-win64пјҢcmakeпјҢpythonпјҢcondaпјҢgitзӯүзҺҜеўғеҸҳйҮҸеҹәжң¬е°ұйқ Scoopзҡ„shimsпјү

жҲ‘д»¬е…ҲеңЁTerminalдёӯиҫ“е…Ҙscoop -vзңӢзңӢscoopе‘Ҫд»ӨжҳҜеҗҰеҸҜд»ҘдҪҝз”Ё

еҸҜд»ҘзңӢеҲ°жҲ‘иҝҷиҫ№еҸҜд»ҘжӯЈеёёжҳҫзӨәзүҲжң¬еҸ·
